# Расширение системы (мета)

> Ты здесь, когда нужно **расширить саму систему разработки** — добавить новое направление-колонку
> (например, мобильную разработку) или новый справочник. Делается ДВУМЯ отдельными сеансами Claude,
> которые владелец запускает по очереди (состояние живёт в памяти, не в одном контексте).

| Шаг | Файл | Роль |
|---|---|---|
| 1. Сборка | [`column-builder.md`](column-builder.md) | Окно-**сборщик**: разведка рынка → рубрика → extract-clean → кладёт колонку в `staging` → отдаёт компактный манифест |
| 2. Приёмка | [`orchestrator.md`](orchestrator.md) | Окно-**оркестратор**: проверяет манифест субагентом → после ревью владельца вносит в `~/.claude/` → обновляет память |
| Справочник | [`dev-system-sources-catalog.md`](dev-system-sources-catalog.md) | Каталог источников рынка — кандидаты для шага «разведка» |

**Порядок:** сборщик (в staging) → ревью владельца → оркестратор (внесение в `~/.claude/`). Сборщик НЕ пишет в `~/.claude/` напрямую. Метод обкатан на 6 колонках (фронт · TS/Node · Python · БД · AI · бэк).

Назад в оглавление: [`../CLAUDE.md`](../CLAUDE.md)
